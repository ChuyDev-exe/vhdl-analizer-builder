# Términos de Servicio

**Última actualización: 2024**

## 1. Aceptación de los Términos

Al utilizar la instancia en línea del **Simulador de Circuitos Lógicos** ("el Servicio"), aceptas estos Términos de Servicio. Si no estás de acuerdo, no utilices el Servicio.

## 2. Descripción del Servicio

El Servicio es una aplicación web del lado del cliente que permite diseñar y simular circuitos lógicos. La aplicación se ejecuta íntegramente en el navegador del usuario; los datos no se almacenan en servidores externos.

Los mantenedores no garantizan que el Servicio sea ininterrumpido, libre de errores o seguro.

- No utilizar el Servicio para actividades ilegales o prohibidas por la ley aplicable.
- No intentar vulnerar la seguridad del Servicio, incluyendo ataques de inyección, denegación de servicio o ingeniería inversa del backend (si lo hubiera).
- No utilizar el Servicio para almacenar o transmitir contenido malicioso.
- El usuario es responsable de sus propios circuitos y datos. Se recomienda realizar copias de seguridad periódicas.
