# Soporte

## Cómo Obtener Ayuda

### GitHub Issues

- **Bugs y reportes de error**: abre un [*issue*](https://github.com/ANOMALYCO/ota-requester-worker/issues/new) y describe el problema con el máximo detalle posible.
- **Solicitudes de funcionalidad**: abre un *issue* con la etiqueta `enhancement`.

### GitHub Discussions

- **Preguntas generales**: usa [GitHub Discussions](https://github.com/ANOMALYCO/ota-requester-worker/discussions) para preguntas, dudas de uso o debates abiertos.
- **Ideas y propuestas**: las *discussions* son el lugar adecuado para conversar antes de crear un *issue* formal.

## Antes de Publicar

1. **Revisa los *issues* existentes** — busca si tu problema ya fue reportado o está en discusión.
2. **Incluye pasos de reproducción** — cuantos más detalles proporciones, más fácil será ayudarte.
3. **Especifica tu entorno** — navegador, sistema operativo y versión del proyecto.

## SLA

Este proyecto es mantenido por voluntarios en su tiempo libre. **No existe un Acuerdo de Nivel de Servicio (SLA)** garantizado. Los tiempos de respuesta varían según la disponibilidad de los mantenedores.

Agradecemos tu paciencia y comprensión.
